### 🔧 Backend - الخيارات

**الخيار 1: Railway (سهل) ⭐**
```bash
npm install -g @railway/cli
railway login
cd backend && railway init
railway up
```

**الخيار 2: Render.com (موصى به)**
1. ادخل https://render.com
2. اضغط "New Web Service"
3. ربط GitHub
4. Build: `pip install -r requirements.txt`
5. Start: `uvicorn main:app --host 0.0.0.0 --port 8000`
6. اضغط "Deploy"

**الخيار 3: Vercel**
```bash
cd backend && vercel --prod
```
